# Chart Sizing (S2)

Spectrum 2 Charts automatically adapts visual density to the chart's rendered width. Line weights and other visual properties scale to the appropriate size tier — no configuration required.

---

## Size tiers

| Tier | Width | Stroke width |
|------|-------|-------------|
| S | < 400px | 1.5px |
| M | 400px – 799px | 2px |
| L | ≥ 800px | 3px |

The tier updates in real time as the chart resizes.

---

## Stroke width

Line strokes scale with the size tier across all stroke-based marks, including lines and metric ranges.

| Tier | Stroke width |
|------|-------------|
| S | 1.5px |
| M | 2px |
| L | 3px |
